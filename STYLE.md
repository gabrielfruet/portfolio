# Style Definition: Steph Ango's Website

**Overall Aesthetic: Minimalist Digital Garden / Typographic Brutalism**

The website embraces a **radically minimal, content-first approach** that prioritizes readability and text above all else. It's the opposite of flashy — no hero images, no hero sections, no animations. Just words on a page.

---

## Key Characteristics

### 1. Ultra-Minimal Layout

- Single-column, centered text with generous whitespace
- Maximum content width of ~37rem (~600px) for optimal reading
- No visual clutter — navigation is stripped to just "About" and "Now"

### 2. Typography-Centric

- System font stack: Apple System, Inter, IBM Plex Sans
- Monospace font for code/UI elements
- Clear hierarchy with subtle heading sizes (no massive H1s)
- Comfortable line-height (1.5) for readability

### 3. Warm, Paper-Like Color Palette (Flexoki)

- Light mode: Off-white/paper background (#FFFCF0) with warm black text
- Dark mode: True black with warm gray text
- Carefully curated accent colors for links, code syntax, and highlights
- Subtle, sophisticated feel — not stark contrast

### 4. Intentional Details

- Custom theme toggle (sun/moon) for light/dark mode
- Muted metadata (dates, reading time) in smaller text
- External links have subtle visual indicators (small arrow)
- Clean dividers using typographic "•••" dots

### 5. Functional, Not Decorative

- No images in the main content area
- Minimal navigation
- Topics organized as plain text tags
- Fast, accessible, no JavaScript bloat

---

## Summary

> It's a **digital garden** aesthetic — quiet, thoughtful, warm, and deeply focused on the written word. Think "academic journal meets personal blog" with a touch of Japanese minimalism.

This style has become somewhat iconic among designers and makers in the tech/minimalist community. It shows that you don't need elaborate design to create something beautiful and readable.
